🤖 AI 多平台聊天插件

完整功能介紹 - 讓 AI 成為您最得力的客服夥伴

📖 插件概述

AI 多平台聊天插件是一個功能強大的 WordPress 插件，專為現代網站設計，整合了 AI 智能聊天和多平台社交媒體連接功能。無論您經營的是電商網站、企業官網還是個人博客，這個插件都能為您的訪客提供即時、智能的客服體驗。

🎯 核心特色

  • AI 智能聊天：支援 GPT-4、Claude、Gemini 等主流 AI 模型
  • 多平台整合：一鍵連接 15+ 社交媒體平台
  • 多語言支援：繁體中文、簡體中文、英文完整支援
  • 數據分析：詳細的對話記錄和使用統計
  • 自訂外觀：豐富的樣式選項和位置設定
  • 高效能：優化的 API 處理和快取機制

🛠️ 安裝與設定指南

步驟 1：插件安裝

  1. 下載插件檔案
  2. 上傳到 /wp-content/plugins/ 目錄
  3. 在 WordPress 後台啟用插件
  4. 進入「聊天設定」開始配置

步驟 2：AI 模型設定

// 推薦設定
API 提供商：OpenRouter
API 金鑰：[您的 OpenRouter API Key]
AI 模型：openai/gpt-3.5-turbo (經濟實惠)
        或 openai/gpt-4 (最佳效果)

步驟 3：平台選擇與配置

  1. 選擇要啟用的平台：勾選所需的聊天平台
  2. 填寫連接資訊：
    • WhatsApp：電話號碼 (格式：+886912345678)
    • LINE：LINE ID 或官方帳號 ID
    • Facebook：粉絲頁 URL 或 Messenger 連結
    • Email：客服信箱地址

步驟 4：外觀自訂

  • 選擇聊天按鈕位置
  • 設定按鈕大小和樣式
  • 調整顏色主題
  • 設定顯示文字
