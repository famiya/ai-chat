== Installation ==

**Automatic Installation**

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "AI Multi-Platform Chat"
4. Click "Install Now" and then "Activate"

**Manual Installation**

1. Download the plugin ZIP file
2. Go to Plugins > Add New > Upload Plugin
3. Choose the ZIP file and click "Install Now"
4. Activate the plugin after installation

**Configuration**

1. Navigate to Settings > AI Chat in your WordPress admin
2. Configure your preferred chat platforms with API keys
3. Set up AI integration with OpenRouter.ai or custom API
4. Customize the chat widget appearance and behavior
5. Enable real-time notifications and analytics
6. Save your settings and test the chat functionality

== Frequently Asked Questions ==

= What chat platforms are supported? =

The plugin supports 7 major platforms: AI Chat (built-in), WhatsApp, Facebook Messenger, LINE, WeChat, QQ, and Instagram Direct Messages.

= Do I need API keys for each platform? =

Yes, you'll need to obtain API keys or access tokens from each platform you want to use. The plugin provides detailed setup instructions for each platform.

= Can I customize the chat widget appearance? =

Absolutely! The plugin offers extensive customization options including colors, positioning, size, animations, and responsive behavior.

= Is the AI chat feature free? =

The AI chat requires an API key from OpenRouter.ai or your preferred AI service provider. Costs depend on your chosen provider and usage.

= Does the plugin support multiple languages? =

Yes, the plugin supports 25+ languages with automatic browser language detection and RTL language support.

= Is real-time chat supported? =

Yes, the plugin includes WebSocket support for real-time messaging and notifications.

= Can I export conversation data? =

Yes, the analytics dashboard includes export capabilities for conversation history and statistics.

= Is the plugin GDPR compliant? =

The plugin includes GDPR compliance features and follows WordPress security best practices.
